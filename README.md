# RAG Research Assistant

A step-by-step Python project to learn and build a Retrieval-Augmented Generation (RAG) Research Assistant.

## Goal
This project is being built incrementally to cover major RAG types and concepts:

- Basic / Naive RAG
- Hybrid Retrieval RAG
- Conversational RAG
- Evaluation-driven RAG
- Agentic RAG
- Optional Multimodal RAG

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- OpenAI API
